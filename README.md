# Application de Gestion de Bibliothèque

Projet réalisé avec **Node.js, Express, EJS et PostgreSQL** dans le cadre d’un TP de développement web full-stack.

##  Fonctionnalités

-  Gestion des auteurs (CRUD)
-  Gestion des livres (CRUD)
-  Recherche de livres
-  Détails des auteurs et livres
-  Interface web moderne avec EJS + CSS
-  Base de données PostgreSQL
  
## Technologies utilisées

- Node.js
- Express.js
- EJS (templating engine)
- PostgreSQL
- HTML / CSS
- JavaScript

## Pages principales
 Accueil (Dashboard)
 Liste des auteurs
 Liste des livres
 Ajouter / Modifier / Supprimer
 Recherche de livres
 Interface

## L’application utilise un design simple et moderne avec :

Layout responsive
Navigation claire
Tableaux et cards
CSS personnalisé
